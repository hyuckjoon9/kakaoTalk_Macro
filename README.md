# 카카오톡 매크로

카카오톡 대화창에 메시지를 자동으로 반복 전송하는 데스크톱 매크로 도구입니다.

---

## 설치 방법

```bash
pip install -r requirements.txt
```

> Python 3.8 이상 권장. Windows 환경에서 동작합니다.

---

## 실행 방법

```bash
python app.py
```

또는 PyInstaller로 단독 실행 파일 빌드:

```bash
pyinstaller app.py
```

---

## 사용 방법

### 공통 흐름

1. **카톡 대화창 좌표 얻기** 버튼 클릭 → 버튼 텍스트가 "클릭 대기 중..."으로 변경됨
2. 전송할 카카오톡 대화창을 마우스로 클릭 → 좌표가 자동 입력됨
3. 메시지와 반복 횟수 입력 (최소 1회)
4. **메시지 전송** 버튼 클릭

![좌표 캡처 화면](screenshots/capture.png) <!-- placeholder -->

### 탭 1 — 단일 메시지 전송

동일한 메시지를 지정한 횟수만큼 반복 전송합니다.

![단일 메시지 전송](screenshots/single.png) <!-- placeholder -->

### 탭 2 — 다중 메시지 전송

여러 줄의 메시지를 한 바퀴에 순서대로 전송합니다. 줄바꿈으로 메시지를 구분합니다.

```
안녕하세요
오늘도 좋은 하루 되세요
감사합니다
```

위와 같이 입력하면 3개 메시지가 순서대로 전송되며, 반복 횟수만큼 반복합니다.

![다중 메시지 전송](screenshots/multi.png) <!-- placeholder -->

### 상태바

전송 중 하단 상태바에서 진행 상황을 확인할 수 있습니다.

| 상태 | 의미 |
|------|------|
| 대기 중 | 전송 준비 완료 |
| 전송 중... (3/10) | 10회 중 3번째 전송 중 |
| 완료 | 모든 전송 완료 |
| 중단됨 | 스페이스바로 중단됨 |

---

## 주의사항

- **스페이스바로 즉시 중단**: 전송 중 스페이스바를 누르면 현재 루프를 완료하지 않고 즉시 중단됩니다.
- **전송 중 GUI 동작**: 전송은 별도 스레드에서 실행되어 전송 중에도 UI가 응답합니다. 단, 전송 버튼은 전송이 완료되거나 중단된 후에만 다시 활성화됩니다.
- **좌표 재캡처**: 카카오톡 창 위치가 바뀌면 좌표 얻기를 다시 진행하세요.
- **설정 변경**: 전송 딜레이, 반복 횟수 기본값 등은 `config.py`에서 수정할 수 있습니다. (`config.example.py` 참고)

---

## 설정 (`config.py`)

`config.example.py`를 복사해 `config.py`를 만든 후 값을 채워 사용합니다.

```bash
cp config.example.py config.py
```

| 항목 | 설명 | 기본값 |
|------|------|--------|
| `SEND_DELAY` | 메시지 전송 간격 (초) | 0.1 |
| `FOCUS_DELAY` | 대화창 포커스 후 대기 시간 (초) | 0.1 |
| `HOTKEY_INTERVAL` | Ctrl+V 단축키 입력 간격 (초) | 0.01 |
| `DEFAULT_REPEAT_SINGLE` | 단일 전송 기본 반복 횟수 | 2 |
| `DEFAULT_REPEAT_MULTI` | 다중 전송 기본 반복 횟수 | 1 |
| `WINDOW_TITLE` | 창 제목 | - |
| `WINDOW_SIZE` | 창 크기 | 800x700 |
